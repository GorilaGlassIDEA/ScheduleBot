# Бот для составления расписания на неделю

Присылаешь список занятий с часами в неделю — бот раскидывает их по вечерам
будней (после 17:00) и по выходным.

## Запуск на сервере (docker compose)

1. Создай бота у [@BotFather](https://t.me/BotFather) командой `/newbot` и скопируй токен.
2. На сервере:
   ```bash
   git clone <репозиторий> && cd бот
   ./set-token.sh        # вставь токен, ввод скрыт
   docker compose up -d --build
   ```

Токен хранится в `secrets/bot_token` (папка с правами 700, в git не попадает) и
передаётся в контейнер как docker-секрет — его не видно ни в
`docker inspect`, ни в переменных окружения. Настройки пользователей
лежат в docker-томе `botdata` и переживают пересборку контейнера.

Полезные команды:

```bash
docker compose logs -f          # логи
docker compose restart          # перезапуск
git pull && docker compose up -d --build   # обновление
```

## Запуск локально (без докера)

1. Создай файл `.env` рядом с `bot.py` (пример — в `.env.example`):
   ```
   BOT_TOKEN=123456:ABC-DEF...
   ```
2. Установи зависимости и запусти:
   ```bash
   pip install -r requirements.txt
   python bot.py
   ```

## Использование

Пришли боту сообщение в формате «название — часы в неделю», каждое занятие
с новой строки:

```
Программирование 10
Английский 5
Спорт 4.5
```

Бот вернёт расписание по дням. Понимает и варианты вроде
`Английский — 5 ч` или `Спорт: 4,5`.

## Настройки (/settings)

- **Будни с …** — с какого времени планировать в будни (по умолчанию 17:00);
- **Выходные с …** — начало дня в субботу и воскресенье (по умолчанию 11:00);
- **Конец дня** — до какого времени планировать (по умолчанию 23:00);
- **Макс. блок** — самый длинный непрерывный отрезок одного занятия (2 ч);
- **Перерыв** — пауза между занятиями (15 мин);
- **Выходные вкл/выкл** — планировать ли субботу и воскресенье;
- **Дни отдыха** — нижний ряд кнопок Пн–Вс: помеченные 🌴 дни бот не трогает.

После изменения настроек кнопка «Пересоставить» строит расписание заново
по последнему присланному списку.

## Как распределяются часы

- часы округляются до получасовых слотов;
- нагрузка делится по дням пропорционально свободному времени в каждом дне;
- внутри дня занятия чередуются, один блок не длиннее «макс. блока»;
- если всё не помещается, бот урежет часы пропорционально и предупредит.
